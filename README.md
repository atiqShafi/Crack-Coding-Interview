### 目录
- Algorithm **包含了数论、树、动态规划**
- Basic **包含了C++基本知识的案例、Boost库的使用**
- CPP-Primer **整理了本书各章节相关的例子**
- CrackInterview
- DesignPattern **包含了每类设计模式的测试代码以及说明**
- Leetcode **包含了leetcode上230多道题目的题解、以及测试用例，覆盖了各类题型**
- POJ
- PythonMachineLearning **机器学习的案例代码**
